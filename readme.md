<ul>

    <li>Update name/username/bio</li>
    <li>Create Posts</li>
    <li>Like/Comment on posts</li>
    <li>Profile view</li>
    <li>Follow/Unfollow other accounts</li>
    <li>Home feed based on accounts you follow</li>
